# Prompt para Imagem do Artigo LinkedIn - Cara Core Dental MVP

## Especificações Técnicas
- **Formato:** 1920x1080 pixels (16:9 - LinkedIn cover)
- **Resolução:** Alta qualidade para redes sociais
- **Orientação:** Horizontal (landscape)

## Prompt Principal

```
Create a professional LinkedIn cover image (1920x1080) for a dental clinic management system MVP article. CRITICAL: ALL VISIBLE TEXT MUST BE IN BRAZILIAN PORTUGUESE.

**Main Elements:**
- **Left side (60%):** Modern dental office interior with clean, minimalist design showing:
  - Contemporary dental chair in light blue/white
  - Digital screens displaying Brazilian Portuguese patient management interface with labels like "Pacientes", "Agendamentos", "Consultas"
  - Professional dental equipment with subtle technology integration
  - Calendar showing Portuguese month names (Janeiro, Fevereiro, etc.)
  - Soft, professional lighting

**Right side (40%):** Technology overlay with:
  - Semi-transparent code snippets in Java/Spring Boot with Portuguese comments floating elegantly
  - AI neural network visualization (subtle, geometric patterns)
  - Small icons representing: calendar, user management, security shield, database
  - Gradient overlay transitioning from medical blue to tech blue

**Typography area (center-bottom):**
- Space for title: "MVP Odontológico: Desenvolvedor + IA"
- Subtitle area: "Cara Core Dental - Transformando a Gestão Clínica"

**IMPORTANT - Text in Portuguese:**
- All visible text elements must be in Portuguese (Brazil)
- Interface elements should show Portuguese labels: "Agendamentos", "Pacientes", "Consultas"
- Any visible code comments should be in Portuguese
- Dashboard/calendar interface should display Portuguese month names and labels
- Buttons should show: "Agendar", "Salvar", "Cancelar", "Confirmar"
- EXCEPTION: Technology names can remain in English (Spring Boot, PostgreSQL, Java, Docker, etc.)

**Color Palette:**
- Primary: Medical blue (#2E86AB, #A23B72)
- Secondary: Clean white (#FFFFFF)
- Accent: Technology green (#00D4AA)
- Background: Soft gradient (light blue to white)

**Style:**
- Modern, professional, clean
- Subtle depth with soft shadows
- Technology meets healthcare aesthetic
- LinkedIn-appropriate business tone
- High contrast for text readability

**Technical details to include:**
- Spring Boot logo (small, corner)
- PostgreSQL elephant icon (subtle)
- Java coffee cup icon (minimal)
- Stethoscope subtly integrated with code brackets

**Mood:** Innovation, professionalism, collaboration between human expertise and AI, healthcare technology advancement.
```
